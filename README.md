# Rales Engine
In this project, I used Rails and ActiveRecord to build a JSON API which exposes the SalesEngine data schema.

## System Requirements:
* Ruby version: 5.1.7
* PostgreSQL
* Ruby 2.4.1

## Gems Used
* RSpec-rails
* Capybara
* Factory_bot_rails
* Selenium-webdriver
* Shoulda-matchers
* Simplecov
* Pry
* Fast_jsonapi
* Faker

## Setup
- run 'rake import:seed'
